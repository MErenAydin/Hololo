# Hololo
Hololo is a baker GUI for specific hardware whichs aim is displaying holographic objects. Using OpenGL and SDL2, objects can be placed, modified and be prepared to render holographicly in render area.

## Requirements
* sdl2
* moderngl
* numpy
* pyrr
* pillow
* pyassimp
